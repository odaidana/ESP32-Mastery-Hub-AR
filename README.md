# مستودع إتقان الـ ESP32: الدليل الشامل للهاردوير والبرمجيات التضمينية

مرحباً بك في هذا المستودع التعليمي المفتوح المصدر والمخصص لاحتراف برمجة وتطوير بوردات الـ ESP32. تم بناء هذا المشروع ليكون مرجعاً خطوة بخطوة يبدأ معك من الصفر وحتى التعامل المتقدم مع العتاد (Hardware) وكتابة الكود الصافي (Firmware).

## الدروس والمصنفات الحالية
1. **الدرس الأول: GPIO & Digital I/O** — التحكم بالمنافذ الرقمية، قراءة الأزرار، ومعالجة تذبذب الإشارة (Debouncing).
2. **الدرس الثاني: PWM Signal Generation** — توليد الإشارات التماثلية للتحكم في شدة الإضاءة ومحركات السيرفو.
3. **الدرس الثالث: Analog Inputs (ADC)** — قراءة الحساسات التناظرية وتحليل الجهد الكهربائي برمجياً.

## هيكلة المجلدات
- `basics/`: يحتوي على الأكواد الأساسية والمباشرة لتشغيل الميزات الحيوية للمعالج.
